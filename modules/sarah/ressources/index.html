Le plugin arret permet d'arret ou de redemarrer l'ordinateur en utilisant les commandes vocales de S.A.R.A.H.

Les commandes vocales sont définies dans le fichier de grammaire XML arret.xml :

  • Sarah éteint le home cinema
  • Sarah redemarre le home cinema
  • Sarah met en veille le home cinema

Sur correspondance, le script arret.js est ensuite exécuté.

Enregistrer bien vos documents en cours, l'action est irréversible !
